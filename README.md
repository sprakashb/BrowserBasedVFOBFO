# BrowserBasedVFOBFO
Control Ham Rig using ESP32 with Si5351 through a browser based Contoller like from a Smart Phone. Initially done with uBITX as model application. Should be possible to control many homebrew rigs.

Details on uploading and connecting to ESP32 via WiFi are in the manual (smvfo*.pdf).

This is purely an experimenters delight. Improve it and share with the community.

One Possible improvement is to transmit voice also using WiFi.
